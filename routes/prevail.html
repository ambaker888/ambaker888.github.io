Prevail

Top Gun Sales Performance, focused on sales training, wanted to make a learning management platform to help the companies with their sales teams.
Objective
  • Build a platform that allowed users to add their own or someone else's content.
  • Create a course management system that any user could easily customize.
  • Enable users to keep track of progress by an individual and a team.
Process
When starting this project I met several times with the stakeholders to get an idea of the business goals and who the target audience was for the application. I also wanted to get an idea of their perception about the competition in the market.

My next step was to conduct a problem discovery test for the existing application. This was conducted with three people in a moderated environment. Based on the results of this test, we were able to fix major problems with the existing application and create several new features.

Once the redesign of the application got started every feature was tested with an A/B test and some form of comparative competition test.
Roles
In addition to user research and usability testing, I also did the design. Most of the mockups were done in HTML and CSS on put on a Heroku instance. There was already a style guide in place with some Angular components; I was able to use this for my javascript needs.
